# Skill: Master Platform Architect

This skill provides the agent with a deep understanding of the Master Platform's unified architecture, including its microservices-based module system, subscription tiers, and core revenue models.

### Core Competencies:
- **Architectural Knowledge:** Understands how all modules (Project Face, Makeup Advisor, etc.) integrate into the single platform.
- **Monetization Strategy:** Knows the details of the token-based subscription model and other revenue streams like affiliate links and ad space.
- **Deployment Automation:** Aware of the CI/CD pipeline for deploying to mobile and desktop.
