# Audrey Evans — Official Hub Site

Personal hub website for **Audrey Evans** — Inventor, Engineer, Advocate, Musician.

## Overview

This is the central "About Me" hub site for Audrey Evans, designed to be pointed at `audreyevansofficial.com` or a hub domain. Individual apps (Project Face, TheAltText, Universal Data Router, etc.) will live in their own separate repositories.

## Tech Stack

- **React 19** + **TypeScript**
- **Tailwind CSS 4** with custom warm amber/orange theme
- **Vite** for build tooling
- **Framer Motion** for animations
- Static site (no backend required)

## Design

- **Theme:** Warm amber/orange glassmorphism dark theme — zero blue light
- **Accessibility:** WCAG AAA compliant, screen reader compatible, neurodivergent-friendly
- **Visual-only notifications** (no audio dependency)
- **Carbon-efficient** dark theme for OLED power savings
- **Mobile-first** responsive design

## Sections

1. **Hero** — Photo, name, tagline, brief intro
2. **About Me** — Full biography
3. **Resume/CV** — Interactive timeline, skills, certifications
4. **Portfolio** — App cards linking to standalone deployments
5. **Music** — Revvel / Hailstorm music brand
6. **Advocacy** — Freedom Angel Corps, trafficking awareness
7. **Contact** — Email, LinkedIn, GitHub, TikTok

## Development

```bash
pnpm install

```

```bash

```

## Deploy

Static output in `dist/public/` — deploy to Vercel, Netlify, or GitHub Pages.
