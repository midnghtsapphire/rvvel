# Project Face: AI Skin Analysis App

**Author:** Audrey Evans

This project contains the backend source code for Project Face, an AI-powered skin analysis and recommendation application.

### Tech Stack
- **Backend:** FastAPI (Python)
- **AI Models:** OpenRouter API
- **Core Logic:** `analysis_engine.py`, `recommendations.py`

### How to Run
1.  Install dependencies: `pip install fastapi uvicorn python-multipart`
2.  Navigate to the `src` directory.
3.  Run the server: `uvicorn main:app --reload`
4.  The API will be available at `http://127.0.0.1:8000`.
