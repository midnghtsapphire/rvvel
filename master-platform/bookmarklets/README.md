# Bookmarklets: One-Click Tools for the Master Platform

**Author:** Audrey Evans
**Date:** 2026-02-15

> Bookmarklets are lightweight JavaScript snippets saved as browser bookmarks. They provide instant access to platform features without installing browser extensions.

## Why Bookmarklets Instead of Extensions?

- **No Store Approval:** No waiting for Chrome Web Store or Firefox Add-ons approval.
- **No Permissions:** No security concerns about granting broad permissions to an extension.
- **One-Click Install:** Just drag the link to your bookmarks bar.
- **Works Everywhere:** Compatible with Chrome, Firefox, Safari, Edge.

## Available Bookmarklets

### 1. Accessibility Analyzer
**File:** `accessibility_analyzer.js`

Scans the current page for common accessibility issues (missing alt text, empty links) and logs them to the console.

### 2. Affiliate Linker
**File:** `affiliate_linker.js`

Automatically adds your Amazon affiliate tag (`audreyevans-20`) to all Amazon links on the current page.

### 3. Skin Analyzer
**File:** `skin_analyzer.js`

Analyzes the first image on the page using the Project Face API and displays skin analysis results.

## How to Install

1.  **Show your bookmarks bar:**
    -   Chrome/Edge: Press `Ctrl+Shift+B` (Windows) or `Cmd+Shift+B` (Mac).
    -   Firefox: Press `Ctrl+Shift+B` (Windows) or `Cmd+Shift+B` (Mac).
    -   Safari: Go to `View > Show Favorites Bar`.

2.  **Create a new bookmark:**
    -   Right-click on your bookmarks bar and select "Add page" or "Add bookmark".

3.  **Name it:**
    -   Give it a descriptive name like "Accessibility Analyzer".

4.  **Paste the code:**
    -   Open the corresponding `.js` file (e.g., `accessibility_analyzer.js`).
    -   Copy the entire JavaScript code.
    -   Paste it into the "URL" or "Location" field of the bookmark.

5.  **Save and use:**
    -   Click the bookmarklet while on any webpage to activate it.
